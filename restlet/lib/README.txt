There are three sets of jars in this directory. 

1. The core API and API implementation:

   com.noelios.restlet.jar
   org.restlet.jar

2. The simple web server for HTTP/HTTPS protocols:

   com.noelios.restlet.ext.simple_3.1.jar
   org.simpleframework.jar

3. The asyncweb web server for HTTP/HTTPS protocols:

   com.noelios.restlet.ext.asyncweb_0.8.jar
   edu.oswego.cs.dl.util.concurrent.jar
   org.apache.commons.logging.jar
   org.apache.mina.jar
   org.safehaus.asyncweb.jar
   org.slf4j.jar
