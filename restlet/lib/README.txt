There are two sets of jars in this directory. 

1. The core API and API implementation:

   com.noelios.restlet.jar
   org.restlet.jar

2. The Jetty web server for HTTP/HTTPS protocols:

   javax.servlet.jar
   com.noelios.restlet.ext.jetty_6.1.jar
   org.mortbay.jetty.ajp.jar
   org.mortbay.jetty.https.jar
   org.mortbay.jetty.jar
   org.mortbay.jetty.util.jar

You can use other server connectors or run this application in a J2EE 
servlet container.  For more information on how to do that with Restlet,
see:

    http://www.restlet.org
